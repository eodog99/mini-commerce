# 🛒 Mini Commerce (React 쇼핑몰 프로젝트)

## 프로젝트 소개

상품 목록 조회부터 장바구니 관리까지 구현한 간단한 쇼핑몰 웹 애플리케이션입니다.
React를 기반으로 컴포넌트 구조 설계와 상태 관리를 중심으로 개발하였습니다.

---

## 개발 목적

* React 컴포넌트 기반 설계 이해
* 전역 상태 관리 경험 (장바구니 기능)
* API 데이터를 활용한 비동기 처리 학습
* 실제 서비스와 유사한 구조 경험

---

## 🛠 기술 스택

### Frontend

* HTML5
* SCSS
* JavaScript (ES6+)
* React

### 상태 관리

* Zustand (또는 Context API)

### 데이터

* Fake API (상품 데이터 사용)

---

## 📂 프로젝트 구조

```
src/
 ├── components/       # 재사용 컴포넌트
 ├── pages/            # 페이지 단위 컴포넌트
 ├── store/            # 상태 관리 (cart)
 ├── styles/           # SCSS 파일
 ├── api/              # API 호출 함수
 └── App.jsx
```

---

## 주요 기능

### 1. 상품 목록 조회

* API를 통해 상품 데이터 가져오기
* 상품 카드 UI 구성

### 2. 상품 상세 페이지

* 상품 클릭 시 상세 페이지 이동
* 상세 정보 표시

### 3. 장바구니 기능

* 상품 추가 / 삭제
* 수량 증가 / 감소
* 총 금액 계산

---

## 데이터 흐름

1. API로 상품 데이터 요청
2. 컴포넌트에서 데이터 렌더링
3. 장바구니 상태는 전역 상태로 관리
4. 여러 컴포넌트에서 상태 공유


## 실행 방법

```bash
npm install
npm run start
```
